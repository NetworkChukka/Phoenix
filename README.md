# Installation

## Android (Termux)

```
pkg update
pkg upgrade
pkg install git wget php
git clone https://github.com/isuruwa/Phoenix
cd Phoenix
chmod +x setup
./setup
```

## Debian (Kali,Parrot)

```
apt update
apt install php wget
git clone https://github.com/isuruwa/Phoenix
cd Phoenix
chmod +x setup
sudo ./setup
```
